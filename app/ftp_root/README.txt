Welcome to Simple FTP Server!

This is a test file for FTP upload and download operations.

You can use any FTP client to connect to this server:
- Host: Server IP address
- Port: 21
- Username: any
- Password: any (not validated)

Supported commands:
- USER: Set username
- PASS: Set password
- SYST: Get system type
- PWD: Print working directory
- CWD: Change working directory
- LIST: List directory contents
- RETR: Download file
- STOR: Upload file
- TYPE: Set transfer type (ASCII/Binary)
- PASV: Enter passive mode
- QUIT: Disconnect

Have fun testing!
